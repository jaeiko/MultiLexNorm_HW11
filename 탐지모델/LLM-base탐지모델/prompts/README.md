프롬프트 구조는 base + (fewshot예제) + regular + input 구조로 하겠습니다

base = 기본적인 작업에 대한 설명 프롬프트입니다
대략 [이 작업은 lexical normalization이다. 문법을 고치고 어쩌고저쩌고를 하면 된다] 정도의 내용이 들어갈 것 입니다

fewshot예제 = 예제입니다
파서옵션이나 config 파일을 통해 예제 선정 방식과 그 개수를 조절할 수 있도록 하겠습니다

regular = output 형식에 대한 지침
output은 "1 [사유]" 또는 "0 [사유]" 둘중 하나라는 내용을 적을 것 입니다. 몇번 돌려보고 잘 지켜지지 않을 경우 더욱 강조해서 적을 계획입니다

input = 입력으로 넣어줄 문자열입니다

종합하면 대략 이정도 프롬프트가 나올 것입니다.
